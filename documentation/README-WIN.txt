 - how to set up and use the wacNetworkMIDI application



Installation & Running
----------------------

No installation is required; the application will run standalone

When first run, Windows will warn about the application trying
gain network access.
You must choose 'unblock' to allow the application access
through Windows' firewall for proper operation.
